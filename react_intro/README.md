## 📝 Project Description

Your **first project with React**! Learn the fundamentals of React including components, JSX syntax, the Virtual DOM, and how to create your first React application from scratch.

### 🎯 Learning Objectives

- Understand what React is and why use it
- Create React elements and components
- Work with JSX syntax
- Understand the Virtual DOM concept
- Set up a React development environment
- Use React Developer Tools
- Write basic tests with Enzyme and Jest

---

## 🚀 Technologies Used

- **React** - UI library for building interfaces
- **JSX** - JavaScript syntax extension
- **Enzyme** - React testing utility
- **Jest** - JavaScript testing framework
- **Webpack** - Module bundler
