## 📝 Project Description

Deep dive into **Advanced React Components**. Master state management, component lifecycle methods, Higher Order Components (HOC), Pure Components, and performance optimization with `shouldComponentUpdate`.

### 🎯 Learning Objectives

- Manage component state with `setState`
- Understand and use component lifecycle methods
- Work with event handlers and controlled components
- Create Higher Order Components (HOC) for code reuse
- Optimize performance with Pure Components
- Use `shouldComponentUpdate` to prevent unnecessary re-renders
- Test stateful components and lifecycle methods
