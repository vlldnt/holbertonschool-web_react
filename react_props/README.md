## 📝 Project Description

Master **Props in React** - the mechanism for passing data between components. Learn how to create reusable components, validate props with PropTypes, and implement component composition patterns.

### 🎯 Learning Objectives

- Pass data between components using props
- Validate props with PropTypes
- Set default props values
- Understand props vs state
- Create reusable and composable components
- Implement component composition patterns
- Use fragments to avoid extra DOM nodes

---

## 🚀 Technologies Used

- **React** - UI library
- **PropTypes** - Runtime type checking for props
- **Jest** - Testing framework
- **Enzyme** - React component testing
