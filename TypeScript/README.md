## 📝 Project Description

This project introduces **TypeScript**, a strongly typed superset of JavaScript that compiles to plain JavaScript. Learn how to add type safety to your code, use interfaces, classes, and leverage TypeScript's powerful type system to catch errors at compile time.

### 🎯 Learning Objectives

- Understand TypeScript's type system (basic types, interfaces, enums)
- Work with classes and inheritance in TypeScript
- Use generic types for reusable components
- Configure TypeScript compiler (`tsconfig.json`)
- Integrate TypeScript with Node.js projects
- Write type-safe code that prevents runtime errors

---

## 🚀 Technologies Used

- **TypeScript** - Static type checking
- **Node.js** - JavaScript runtime
- **ts-node** - TypeScript execution engine
- **npm** - Package management
