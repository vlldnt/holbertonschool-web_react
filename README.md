## 📚 Projects Overview

This repository contains all projects from the **React Web Development** specialization at Holberton School. Each project builds on the previous one, covering essential concepts in modern web development.

| 📂 Project | 🎯 Description | 🛠️ Technologies |
|-----------|---------------|-----------------|
| [**TypeScript**](./TypeScript) | Introduction to TypeScript - static typing, interfaces, classes, and type safety in JavaScript applications | TypeScript, Node.js |
| [**Webpack**](./Webpack) | Modern JavaScript bundling - configuring Webpack, code splitting, asset management, dev server, and optimization | Webpack, Babel, CSS Loaders |
| [**React Intro**](./react_intro) | First steps with React - components, JSX, virtual DOM, and creating your first React application | React, JSX, Enzyme |
| [**React Props**](./react_props) | Component communication - passing data with props, PropTypes validation, and component composition patterns | React, PropTypes, Jest |
| [**React Component**](./react_component) | Advanced components - state management, lifecycle methods, HOC (Higher Order Components), and Pure Components | React, Enzyme, Jest |
| [**React Styling**](./react_styling) | Styling in React - CSS Modules, inline styles, CSS-in-JS, Aphrodite, and responsive design | React, Aphrodite, CSS |

---

## 🎓 Learning Objectives

Throughout these projects, you will master:

- ✅ **TypeScript** - Strong typing and modern JavaScript features
- ✅ **Webpack** - Module bundling and build optimization
- ✅ **React Fundamentals** - Components, JSX, Virtual DOM
- ✅ **Props & State** - Data flow and component communication
- ✅ **Component Lifecycle** - Managing component behavior
- ✅ **Testing** - Jest and Enzyme for React applications
- ✅ **Styling** - Modern CSS approaches in React
- ✅ **Performance** - Optimization techniques and best practices
