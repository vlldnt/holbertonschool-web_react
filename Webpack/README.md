## 📝 Project Description

Learn **Webpack**, the powerful module bundler for modern JavaScript applications. This project covers everything from basic configuration to advanced optimization techniques like code splitting, lazy loading, and asset management.

### 🎯 Learning Objectives

- Configure Webpack from scratch (`webpack.config.js`)
- Set up development and production builds
- Use loaders for CSS, images, and other assets
- Implement code splitting and lazy loading
- Configure Webpack Dev Server with Hot Module Replacement
- Optimize bundle size and performance
- Integrate Babel for modern JavaScript transpilation

---

## 🚀 Technologies Used

- **Webpack** - Module bundler
- **Babel** - JavaScript transpiler
- **webpack-dev-server** - Development server with HMR
- **CSS Loaders** - Style processing
- **File Loaders** - Asset management
