## Project Description

Deep dive into **React State Management**. Master local state, controlled components, React Hooks, and the Context API to manage and share data across your component tree.

### Learning Objectives

- Understand what the state of a component or a container is
- Master the lifecycle of a component
- Modify a state and execute code in the right order
- Work with controlled components and Forms in React
- Reuse smaller components, keep them pure, and lift state to principal containers
- Use React Hooks and create custom ones
- Pass data deeply through the component tree using Context

---

## Technologies Used

- **React** - UI library with class & functional components
- **Jest** - Testing framework
- **Enzyme** - Component testing utilities
- **React Context API** - Data sharing without prop drilling
