## 📝 Project Description

Master **React Hooks** - the modern way to use state and other React features in functional components. Learn how to manage component state with `useState`, handle side effects with `useEffect`, consume context with `useContext`, and build custom hooks for reusable logic.

### 🎯 Learning Objectives

- Use the appropriate hooks for state management and side effects
- Properly consume context using the `useContext` hook
- Maintain clean and efficient component structure
- Handle authentication flows with hooks
- Display dynamic content based on application state
- Manage component lifecycle with hooks
- Ensure no console errors or warnings
- Write maintainable, lint-free code

---

## 🔧 Hooks Covered

- **`useState`** - Manage local component state
- **`useEffect`** - Handle side effects and lifecycle
- **`useContext`** - Consume context values from providers
- **Custom Hooks** - Create reusable logic across components

## 🚀 Technologies Used

- **React** - UI library with hooks
- **React Context API** - State management
- **Jest** - Testing framework
- **Enzyme** - React component testing
- **PropTypes** - Runtime type checking (optional)
